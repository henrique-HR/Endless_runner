# 🎮 Runner Game

> Endless runner criado em 2021 quando estava aprendendo a programar. Revisitado em 2025 com código refatorado, colisão correta e suporte mobile. Do iniciante ao dev que entrega.

---

## 🕹️ Como jogar

- **Desktop:** Pressione `Espaço` ou `↑` para pular
- **Mobile:** Toque no botão **Pular** na tela
- Desvie das pedras e sobreviva o maior tempo possível
- Clique em **▶ Jogar** para iniciar ou reiniciar

---

## ✨ Funcionalidades

- Personagem animado com sprites próprios
- Obstáculos, nuvens, árvore e arbusto em movimento
- Sistema de score em tempo real
- Efeitos sonoros e música de fundo
- Tela de game over com animação
- Botão Jogar com suporte a fullscreen e landscape automático no mobile
- Responsivo para desktop e mobile
